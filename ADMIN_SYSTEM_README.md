# YELO Admin-Only Authentication System

نظام مصادقة محدود للأدمن فقط مع إمكانيات إدارة شاملة

## المميزات الجديدة

### 🔐 **نظام محدود للأدمن**
- الأدمن فقط يستطيع إنشاء المستخدمين
- لا يوجد مستخدمين افتراضيين (عدا الأدمن)
- جميع المستخدمين يتم إنشاؤهم من لوحة الإدارة

### 👤 **إدارة المستخدمين الكاملة**
- إنشاء مستخدمين جدد
- تحديد الصلاحيات لكل مستخدم
- تفعيل/إلغاء تفعيل المستخدمين
- حذف المستخدمين

### 🔑 **إدارة كلمات المرور**
- عرض جميع كلمات المرور (للأدمن فقط)
- تغيير كلمة مرور أي مستخدم
- تغيير كلمة مرور الأدمن
- حفظ كلمات المرور بشكل مرئي

### 📊 **مراقبة شاملة للأنشطة**
- تسجيل جميع محاولات تسجيل الدخول
- تسجيل الوصول للمشاريع
- تسجيل إدارة المستخدمين
- تسجيل تغيير كلمات المرور
- عرض آخر 200 نشاط

## بيانات الدخول الافتراضية

```
اسم المستخدم: admin
كلمة المرور: admin123
```

## كيفية الاستخدام

### 1. تسجيل الدخول كأدمن
1. اذهب لصفحة `login.html`
2. أدخل بيانات الأدمن الافتراضية
3. ستنتقل للوحة الرئيسية

### 2. إدارة المستخدمين
1. اضغط على زر "Admin" في الشريط العلوي
2. ستفتح لوحة الإدارة
3. استخدم نموذج "Add New User" لإضافة مستخدمين جدد

### 3. إدارة كلمات المرور
1. في لوحة الإدارة، اضغط "View All Passwords"
2. لتغيير كلمة مرور مستخدم: اختر المستخدم وأدخل كلمة المرور الجديدة
3. لتغيير كلمة مرور الأدمن: اضغط "Change Admin Password"

### 4. مراقبة الأنشطة
- جميع الأنشطة تظهر في قسم "Activity Log"
- يمكن الضغط على "Refresh" لتحديث القائمة

## الأمان والحماية

### ✅ **المميزات الأمنية**
- فحص الصلاحيات قبل كل وصول
- انتهاء صلاحية الجلسات (24 ساعة)
- تسجيل جميع الأنشطة مع التوقيت
- منع الوصول غير المصرح به
- إعادة توجيه تلقائية للمستخدمين غير المصرح لهم

### 🔒 **حماية البيانات**
- جميع البيانات محفوظة محلياً
- كلمات المرور مرئية للأدمن فقط
- سجل الأنشطة محمي من التلاعب
- فصل صلاحيات الأدمن عن المستخدمين العاديين

## وظائف الأدمن المتاحة

### إدارة المستخدمين:
- `createUser(userData)` - إنشاء مستخدم جديد
- `getAllUsers()` - عرض جميع المستخدمين
- `deleteUser(username)` - حذف مستخدم
- `toggleUserStatus(userId)` - تفعيل/إلغاء تفعيل

### إدارة كلمات المرور:
- `getUserPasswords()` - عرض جميع كلمات المرور
- `updateUserPassword(username, newPassword)` - تغيير كلمة مرور مستخدم
- `updateAdminPassword(newPassword)` - تغيير كلمة مرور الأدمن

### مراقبة الأنشطة:
- `logActivity(user, action, details)` - تسجيل نشاط
- `getActivities()` - عرض سجل الأنشطة

## استكشاف الأخطاء

### مشاكل شائعة:

**لا يمكن تسجيل الدخول:**
- تأكد من صحة اسم المستخدم وكلمة المرور
- تأكد من تحميل ملف `auth-system.js`

**"الوصول مرفوض" للمشروع:**
- تأكد من أن المستخدم له صلاحية للمشروع
- تأكد من أن المستخدم نشط (غير معطل)

**لا تظهر لوحة الإدارة:**
- تأكد من تسجيل الدخول كأدمن
- تأكد من وجود الصلاحيات الصحيحة
